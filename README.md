## Architecture Support

These images are built to be compatible with the following:

1. `linux/amd64`
2. `linux/arm64` (Apple Silicon)

<br>

| Architecture / OS     | Support |
|-----------------------|:--------:|
| macOS (Apple Silicon) | ✅        |
| macOS (Intel Silicon) | ✅        |
| Linux (64 Bit)        | ✅        |
| Windows (64 Bit)      | ✅        |
